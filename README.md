# Port Checker

GUI tool for Windows that shows which processes are using which ports.

- **Port** — TCP & UDP listening ports
- **Process** — PID, Name, Path
- **Resources** — Memory (MB) & CPU %
- **Filter** by port number
- **Kill** process directly
- **Auto-refresh** every 2.5s
- **Sort** any column

Single .exe, no install required. Built with C# WinForms (.NET Framework).
